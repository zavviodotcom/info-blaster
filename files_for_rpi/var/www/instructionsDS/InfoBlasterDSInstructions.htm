1.      Digital Signage Administration

Using the standard configuration, the first step is to connect using wireless device (e. g., laptop). Then enter address http://10.0.0.1/adminDS/adminDS.html using browser (note: System will redirect many addresses, such as http://www.info-blaster.com/adminDS/adminDS.html or http://www.info-blaster.com/adminDS). Upon connection of the browser, you will be prompted for username and password. After initial power-up or after System has been reset to factory settings, both username and password are set as info-blaster.

2.      Configure General/Server Settings

The digital signage feature is incorporated with Info-Blaster. General configuration as well as configuration of the web server can be accessed using the Go to general/web-server admin button. It can also be accessed directly at http://10.0.0.1/admin/admin.html using browser (note: System will redirect many addresses, such as http://www.info-blaster.com/admin/admin.html or http://www.info-blaster.com/admin).

3.      Load New File(s) to Display

The files to be displayed must all be jpg (picture) files. They will be displayed as a slideshow (or a fixed display if only one jpg file is uploaded). First the file(s) to be displayed need to be prepared with any suitable tool. For multiple files, create a compressed folder at the root directory of the file; for a single file, no compression is required. After the pictures have been prepared, connect to digital signage administration page as described in the instructions for administration of the System. Click the Choose File button and browse to the single JPG or compressed (e. g., zip) file. Then click the Submit button. The max file size supported is 1GB, which may take a few minutes to upload.

If the last file(s) uploaded are no longer desired, the Undo/Redo button can be used to revert to the previous file(s) that were uploaded. The System only stores the last 2 sets of file(s) so it is not possible to revert to older files than the previous set; the undo/redo button toggles between the last 2 sets that were uploaded.

By its nature the System is designed for public access of display and web pages. Therefore, it is recommended that you not include confidential or secure information. Also it is your responsibility to comply with applicable regulations, including Copyright laws.

4.      Configure Display Settings

Connect to administration page as described in the instructions for administration of the System. Click the Edit Digital Signage Config button. To change the display settings, edit the line that starts with fbi. Then click the Save button. For example, to change the time between slides, -t 3 could be changed to -t 10.
